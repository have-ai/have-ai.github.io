快速开始 ​

参照官网教程：Get started with Go，10分钟完成started教程。

先决条件 ​

1、编程经验。

2、用于编辑代码的IDE。推荐 VSCode (免费)。

3、命令终端：Windows 可以使用 PowerShell 或 cmd 。

Coding "Hello World" ​

Go已经安装完成，直接进入编码。

bash
mkdir hello
cd hello
go mod init example/hello
type nul > hello.go

在通过编辑器，编辑hello.go，输入以下代码：

go
package main //定义 main package

import "fmt" //引入 fmt package： formatting text, including printing to the console.

func main() {
    fmt.Println("Hello, World!")  
}

运行hello.go，输出Hello，World!

bash
go run .

VSCode + Go ​

在VScode中配置 Go开发环境。

Go语言环境目前已安装，VSCode可以在VS Code官网下载。

安装由"Go Team at Google"发布的Go扩展 ​

VSCode Go开发环境的设置可以参照：VS Code设置，推荐有标准开发配置、性能优化配置、学习环境配置、团队协作配置，各配置项也给出了说明；而且，也推荐了一些必要工具、快捷键和常见问题。

注：通过go install安装第三方包，默认使用的是proxy.golang.org，在国内无法访问，可以切换为国内能访问的代理地址：https://goproxy.cn

bash
go env -w GOPROXY=https://goproxy.cn

这里先选用学习环境配置：

json
{
    "go.useLanguageServer": true,
    "go.formatTool": "goimports",
    "go.lintTool": "golangci-lint",
    "go.lintOnSave": "workspace",
    "go.vetOnSave": "workspace",
    "go.testOnSave": true,
    "go.coverOnSave": true,
    "go.toolsManagement.autoUpdate": true
}

引入第三方包 ​

通过VSCode打开hello文件夹，并编辑引入 rsc.io/quote package。

go
package main //定义 main package

import (
	"fmt"

	"rsc.io/quote"
) //引入 fmt package： formatting text, including printing to the console.

func main() {
	fmt.Println("Hello, World!")
	fmt.Println(quote.Go())
}

Go import需要通过go.sum文件验证模块，执行如下命令：

bash
go mod tidy

最后，在VSCode可通过快捷键F5调试代码。
