基本语法 ​

Go 是一种通用语言，旨在进行系统编程。它具有强类型和垃圾收集功能，并明确支持并发编程。程序由包构建，其属性允许高效管理依赖关系。

类型Types ​

布尔类型（Boolean） ​

bool，只能是常量true/false。 比如：

go
var b bool = false
b := true

数值类型（Numeric） ​

包括整数、浮点数或复数。预先声明的类型包括如下：

go
uint8       the set of all unsigned  8-bit integers (0 to 255)
uint16      the set of all unsigned 16-bit integers (0 to 65535)
uint32      the set of all unsigned 32-bit integers (0 to 4294967295)
uint64      the set of all unsigned 64-bit integers (0 to 18446744073709551615)

int8        the set of all signed  8-bit integers (-128 to 127)
int16       the set of all signed 16-bit integers (-32768 to 32767)
int32       the set of all signed 32-bit integers (-2147483648 to 2147483647)
int64       the set of all signed 64-bit integers (-9223372036854775808 to 9223372036854775807)

float32     the set of all IEEE 754 32-bit floating-point numbers
float64     the set of all IEEE 754 64-bit floating-point numbers

complex64   the set of all complex numbers with float32 real and imaginary parts
complex128  the set of all complex numbers with float64 real and imaginary parts

byte        alias for uint8
rune        alias for int32

字符串类型（String） ​

字符串类型表示字符串值的集合。预声明的字符串类型是 string。

数组类型（Array） ​

数组是单一类型的元素的编号序列，称为元素类型。元素的数量称为数组的长度，且从不为负。如：

go
var num [5]int  //默认所有元数都是0
var num = [5]int{1,2,3,4,5} //初始化
num := [5]int{1,2,3,4,5}  //简短声明
//遍历数组
for i, value := range num {
    fmt.Printf("Index: %d, Value: %d\n", i ,value)
}
var matrix [2][3]int //多维数组
matrix = [2][3]int{{1, 2}, {3, 4, 5}}

数组的长度是固定的，非动态的，要动态可以使用切片(Slice)类型。

切片类型（Slice） ​

可以看作是动态数组，长度可变，是基于数组创建的内置类型。如：

go
s := make([]int, 2)  //创建一个长度为2的整型切片 make([]T, length, capacity)
var slice []int = []int{1, 2}
s[0] = 1
s[1] = 2
s= append(s, 3)   //向切片追加元素

结构体类型（Struct） ​

结构是一系列命名元素，称为字段，每个元素都有名称和类型。如：

go
// An empty struct.
struct {}

// A struct with 6 fields.
struct {
	x, y int
	u float32
	_ float32  // padding
	A *[]int
	F func()
}

指针类型（Pointer） ​

指针类型表示指向给定类型变量的所有指针的集合，称为指针的基本类型。未初始化的指针的值为 nil。

go
*Point
*[4]int

var ptr01 *string = new(string) //new函数可以创建对应类型的指针
ptr01 := new(string) 
*ptr01 = "00000string"

函数类型（Function） ​

函数类型表示具有相同参数和结果类型的所有函数的集合。函数类型的未初始化变量的值为 nil。

go
func()
func(x int) int
func(a, _ int, z float32) bool
func(a, b int, z float32) (bool)
func(prefix string, values ...int)
func(a, b int, z float64, opt ...interface{}) (success bool)
func(int, int, float64) (float64, *[]int)
func(n int) func(p *T)

接口类型（Interface） ​

接口类型定义类型集。接口类型变量可以存储接口类型集中任何类型的值。这种类型被称为实现接口。未初始化的接口类型变量的值为 nil。

可以区分为Basic interfaces、Embedded interfaces、General interfaces

映射类型（Map） ​

映射是一种类型的元素的无序组合，称为元素类型，由另一种类型的一组唯一键 (称为键类型) 索引。未初始化的映射的值为 nil。

go
var scores map[string]int  //声明
scores = make(map[string]int)  //初始化
scores := make(map[string]int)  //声明+初始化
scores := map[string]int {  //声明+赋值初始化
    "a": 100,
    "b": 90,
    "c": 80
}

管道类型（Channel） ​

管道提供了一种并发执行函数的机制，通过发送和接收指定元素类型的值来进行通信。未初始化的信道的值为 nil。

go
var ch chan int // 声明一个int型的chan，值为nil
ch = make(chan int) // 创建一个无缓冲的chan
ch1 := make(chan int, 4) // 创建一个有缓冲的chan，缓冲大小为4
ch <- 1 // 向chan写入数据
val := <-ch // 从chan读取数据并赋值给val
close(ch) // 关闭chan
val, ok := <-ch // 如果ok为true，表示读取到零值；如果ok为false，表示chan已关闭且没有数据

类型别名 ​

主要用于解决代码升级，保证兼容，是类型的别名定义。

go
type TypeAlias = int // TypeAlias 只是int的别名。
type TypeAlias int //定义TypeAlias为int类型，TypeAlias是新类型

语句（statements） ​

IF语句 ​

“If” 语句根据布尔表达式的值指定两个分支的条件执行。如果表达式的计算结果为 true, 则执行 “if” 分支；否则，如果存在，则执行 “else” 分支。

go
if x := f(); x < y {
	return x
} else if x > z {
	return z
} else {
	return y
}

Switch语句 ​

“switch” 语句提供多种执行方式。将表达式或类型与 “switch” 内的 “cases” 进行比较，以确定要执行的分支。

go
switch tag {
default: s3()
case 0, 1, 2, 3: s1()
case 4, 5, 6, 7: s2()
}

switch x := f(); {  // missing switch expression means "true"
case x < 0: return -x
default: return x
}

switch {
case x < y: f1()
case x < z: f2()
case x == 4: f3()
}

switch i := x.(type) {
case nil:
	printString("x is nil")                // type of i is type of x (interface{})
case int:
	printInt(i)                            // type of i is int
case float64:
	printFloat64(i)                        // type of i is float64
case func(int) float64:
	printFunction(i)                       // type of i is func(int) float64
case bool, string:
	printString("type is bool or string")  // type of i is type of x (interface{})
default:
	printString("don't know the type")     // type of i is type of x (interface{})
}

For语句 ​

“for” 语句指定重复执行一个块。有三种形式：迭代可以由单个条件、“for” 子句或 “range” 子句控制。

go
var prints []func()
for i := 0; i < 5; i++ {
	prints = append(prints, func() { println(i) })
	i++
}

for _, p := range prints {
	p()
}

var key string
var val interface{}  // element type of m is assignable to val
m := map[string]int{"mon":0, "tue":1, "wed":2, "thu":3, "fri":4, "sat":5, "sun":6}
for key, val = range m {
	h(key, val)
}
// key == last map key encountered in iteration
// val == map[key]

var ch chan Work = producer()
for w := range ch {
	doWork(w)
}

Select语句 ​

选择一组可能的发送或接收操作中的哪一个将要进行。它看起来类似于 “开关” 语句，但其中所有情况都指的是通信操作。

go
var a []int
var c, c1, c2, c3, c4 chan int
var i1, i2 int
select {
case i1 = <-c1:
	print("received ", i1, " from c1\n")
case c2 <- i2:
	print("sent ", i2, " to c2\n")
case i3, ok := (<-c3):  // same as: i3, ok := <-c3
	if ok {
		print("received ", i3, " from c3\n")
	} else {
		print("c3 is closed\n")
	}
case a[f()] = <-c4:
	// same as:
	// case t := <-c4
	//	a[f()] = t
default:
	print("no communication\n")
}

for {  // send random sequence of bits to c
	select {
	case c <- 0:  // note: no statement, no fallthrough, no folding of cases
	case c <- 1:
	}
}

首次提问 ​

Q：你是一个go语言的专家，请帮我写一个带布尔类型、数值类型、字符串类型、数组类型、切片类型、结构体类型、指针类型、函数类型、接口类型、映射类型、管道类型、if语句、swith语句、for语句、select语句、send语句的代码学习样例。请直接给代码，并提供足够的注释。

go
// hello.go
package main

import (
	"fmt"
	"time"
)

// 定义一个接口
type Speaker interface {
	Speak() string
}

// 定义一个结构体类型
type Person struct {
	Name string
	Age  int
}

// 为Person实现Speaker接口
func (p Person) Speak() string {
	return fmt.Sprintf("我叫%s, 今年%d岁", p.Name, p.Age)
}

// 定义一个函数类型
type GreetFunc func(string) string

func main() {
	// 1. 布尔类型
	var isGoFun bool = true
	fmt.Printf("布尔类型: %t\n", isGoFun)

	// 2. 数值类型
	var num int = 42
	var pi float64 = 3.14159
	fmt.Printf("整型: %d, 浮点型: %.2f\n", num, pi)

	// 3. 字符串类型
	var greeting string = "Hello, 世界"
	fmt.Printf("字符串: %s\n", greeting)

	// 4. 数组类型 (固定长度)
	var numbers [3]int = [3]int{1, 2, 3}
	fmt.Printf("数组: %v\n", numbers)

	// 5. 切片类型 (动态数组)
	var colors []string = []string{"红", "绿", "蓝"}
	colors = append(colors, "黄") // 添加元素
	fmt.Printf("切片: %v\n", colors)

	// 6. 结构体类型
	var p Person = Person{Name: "张三", Age: 25}
	fmt.Printf("结构体: %+v\n", p)

	// 7. 指针类型
	var ptr *Person = &p
	fmt.Printf("指针: %p, 值: %+v\n", ptr, *ptr)

	// 8. 函数类型
	var greet GreetFunc = func(name string) string {
		return "你好, " + name
	}
	fmt.Println("函数调用:", greet("李四"))

	// 9. 接口类型
	var s Speaker = p
	fmt.Println("接口方法调用:", s.Speak())

	// 10. 映射类型
	var m map[string]int = map[string]int{
		"苹果": 5,
		"香蕉": 3,
	}
	fmt.Printf("映射: %v\n", m)

	// 11. 管道类型
	ch := make(chan string, 1)
	
	// 12. if语句
	if num > 40 {
		fmt.Println("if语句: num大于40")
	} else {
		fmt.Println("if语句: num不大于40")
	}

	// 13. switch语句
	switch num {
	case 42:
		fmt.Println("switch语句: 答案是42")
	case 100:
		fmt.Println("switch语句: 100")
	default:
		fmt.Println("switch语句: 默认情况")
	}

	// 14. for语句
	fmt.Print("for循环: ")
	for i := 0; i < 3; i++ {
		fmt.Printf("%d ", i)
	}
	fmt.Println()

	// 15. select语句 (通常与通道一起使用)
	go func() {
		time.Sleep(100 * time.Millisecond)
		ch <- "来自goroutine的消息"
	}()

	select {
	case msg := <-ch:
		fmt.Println("select语句:", msg)
	case <-time.After(200 * time.Millisecond):
		fmt.Println("select语句: 超时")
	}

	// 16. send语句 (向通道发送数据)
	ch <- "直接发送的消息"
	fmt.Println("send语句:", <-ch)
}

二次提问 ​

Q：生成代码逻辑性不足，帮助强化下。

go
// hello.go
package main

import (
	"fmt"
	"time"
)

// 定义一个接口
type Speaker interface {
	Speak() string
}

// 定义一个结构体类型
type Person struct {
	Name string
	Age  int
}

// 为Person实现Speaker接口
func (p Person) Speak() string {
	return fmt.Sprintf("我叫%s, 今年%d岁", p.Name, p.Age)
}

// 定义一个函数类型
type GreetFunc func(string) string

func main() {
	// 1. 布尔类型 - 添加逻辑运算
	isGoFun := true
	isHard := false
	fmt.Printf("布尔逻辑运算: %t && %t = %t\n", isGoFun, isHard, isGoFun && isHard)

	// 2. 数值类型 - 添加计算
	num := 42
	pi := 3.14159
	circumference := float64(num) * pi
	fmt.Printf("计算: 直径%d的圆周长 = %.2f\n", num, circumference)

	// 3. 字符串类型 - 添加操作
	greeting := "Hello, 世界"
	fmt.Printf("字符串操作: 长度=%d, 首字符=%c\n", len(greeting), greeting[0])

	// 4. 数组类型 - 添加遍历
	numbers := [3]int{1, 2, 3}
	sum := 0
	for _, n := range numbers {
		sum += n
	}
	fmt.Printf("数组遍历求和: %v = %d\n", numbers, sum)

	// 5. 切片类型 - 添加容量检查
	colors := []string{"红", "绿", "蓝"}
	fmt.Printf("切片初始: len=%d cap=%d %v\n", len(colors), cap(colors), colors)
	colors = append(colors, "黄", "紫")
	fmt.Printf("切片扩容后: len=%d cap=%d %v\n", len(colors), cap(colors), colors)

	// 6. 结构体类型 - 添加方法调用
	p := Person{Name: "张三", Age: 25}
	fmt.Printf("结构体方法: %s\n", p.Speak())

	// 7. 指针类型 - 修改原值
	ptr := &p
	ptr.Age++
	fmt.Printf("指针修改: 年龄增加后 %+v\n", *ptr)

	// 8. 函数类型 - 高阶函数
	greet := func(name string) string {
		return "你好, " + name
	}
	formalGreet := func(g GreetFunc, name string) string {
		return "尊敬的" + g(name)
	}
	fmt.Println("高阶函数:", formalGreet(greet, "李四"))

	// 9. 接口类型 - 多态
	var s Speaker = p
	describe(s)

	// 10. 映射类型 - 安全访问
	m := map[string]int{"苹果": 5, "香蕉": 3}
	if qty, exists := m["橙子"]; exists {
		fmt.Println("橙子数量:", qty)
	} else {
		fmt.Println("没有橙子库存")
	}

	// 11. 管道类型 - 带缓冲
	ch := make(chan string, 2)

	// 12. if语句 - 带初始化
	if x := 10; x > num {
		fmt.Printf("if带初始化: %d > %d\n", x, num)
	}

	// 13. switch语句 - 表达式
	switch {
	case num < 0:
		fmt.Println("负数")
	case num > 0 && num < 100:
		fmt.Println("1-99之间的正数")
	default:
		fmt.Println("100或更大的数")
	}

	// 14. for语句 - range遍历
	fmt.Print("for range遍历: ")
	for i, color := range colors {
		fmt.Printf("%d:%s ", i, color)
	}
	fmt.Println()

	// 15. select语句 - 多通道
	ch2 := make(chan int)
	go func() {
		time.Sleep(50 * time.Millisecond)
		ch2 <- 42
	}()

	select {
	case msg := <-ch:
		fmt.Println("从ch收到:", msg)
	case num := <-ch2:
		fmt.Println("从ch2收到:", num)
	case <-time.After(100 * time.Millisecond):
		fmt.Println("select超时")
	}

	// 16. send语句 - 检查通道容量
	if len(ch) < cap(ch) {
		ch <- "新的消息"
		fmt.Println("send成功:", <-ch)
	}
}

// 辅助函数
func describe(s Speaker) {
	fmt.Printf("接口类型: (%v, %T)\n", s, s)
}
